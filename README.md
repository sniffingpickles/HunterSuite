# Hunter Suite

A comprehensive Hunter toolkit for WoW Classic Anniversary Edition. All the tools a Hunter needs in one lightweight addon.

## Features

### Pet Management
- **Pet Feed Bar** - Shows pet happiness with one-click feeding. Auto-selects the best food from your bags.
- **Pet Happiness Alerts** - Visual warnings when your pet is unhappy
- **Mend Pet Reminder** - Alerts when pet health drops below threshold

### Shot Timer
- **Auto Shot Timer** - Precise swing timer with latency compensation
- **Queue Window** - Shows optimal ability queue timing
- **Customizable appearance** - Adjust size, scale, and text labels

### Combat Tools
- **Trap Timer** - Track active trap durations
- **Range Indicator** - Know when you're in range, dead zone, or melee range
- **Serpent Sting Tracker** - Shows sting duration on nameplates
- **Aspect Reminder** - Alert when no aspect is active

### Utility
- **Ammo Warning** - Low ammo alerts with configurable thresholds
- **Growl Reminder** - Warns to disable Growl when entering dungeons/raids
- **Auto-Mark Pets** - Automatically marks enemy Hunter pets with Skull in PvP

## Slash Commands

| Command | Description |
|---------|-------------|
| `/hs` | Open settings panel |
| `/hs edit` | Toggle edit mode to reposition UI elements |
| `/hs lock` | Lock/unlock all bars |
| `/hs feed` | Feed your pet |
| `/hs reset` | Reset all settings to defaults |

## Installation

1. Download and extract to your `Interface/AddOns/` folder
2. Ensure the folder is named `HunterSuite`
3. Restart WoW or type `/reload`

## Configuration

All settings are accessible through the in-game settings panel (`/hs`). Features include:
- Enable/disable individual modules
- Adjust scale and opacity
- Customize alert thresholds
- Position UI elements with Edit Mode
